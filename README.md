# TuPrimeraPagina+Maffei

## Flujo de prueba
1. Crear **Proyectos**: `/proyectos/` → “Nuevo”.
2. Crear **Etiquetas**: `/etiquetas/` → “Nueva”.
3. Crear **Tareas**: `/tareas/` → “Nueva” (asociar proyecto y etiquetas).
4. Ver **Listado de Tareas**: `/tareas/`.
5. **Buscar Tareas**: `/tareas/buscar/` (por título, estado y/o etiqueta).
